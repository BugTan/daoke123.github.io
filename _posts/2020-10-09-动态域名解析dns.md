https://github.com/NewFuture/DDNS

    git clone https://github.com/NewFuture/DDNS
    cd DDNS
    ./run.py #运行一次 生成config.json配置文件
    vi config.json #修改 id 域名 token
    ./task.sh

然后运行命令crontab -e ,将脚本加入定时运行中

    crontab -e
末尾添加一行（复制后到黑色窗口直接鼠标右键就行）

    */5 * * * * /root/DDNS/task.sh
